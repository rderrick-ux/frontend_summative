Student Finance Tracker

A simple expense tracking app built for students to monitor spending habits and stay within budget.

Features
1. Dashboard: View total spent, budget remaining, top categories, and recent activity.
2. Transactions: Search, sort, and manage all expenses.
3. Add Transaction: Record new expenses with validation.
4. Settings: Set monthly budget cap, import/export data as JSON.
5. Responsive: Works on phones, tablets, and desktops

How to Use

1. Go to Add Transaction.
2. Fill in description, amount, category, and date.
3. Click "Add Transaction".

Managing Transactions

1. Search: Type to filter by description.
2. Sort: Click the sort icon or change the field.
3. Edit: Click the pencil icon on any transaction.
4. Delete: Click the trash icon.

Setting a Budget

1. Go to Settings.
2. Enter your monthly spending cap.
3. Click Save.

Import/Export Data

1. Export: Click "Export to JSON" to download your data.
2. Import: Select a JSON file to restore previous data.

Tech Stack

1. Pure HTML, CSS, and JavaScript.
2. Local storage for data persistence.
3. Regex powered validation.
4. Mobile responsive design.
